a标签

定义和用法：

  • <a>：标签定义超链接，用于从当前页面链接（跳转）到另一张页面，是页面中的重要标签之一。现在的网站是通过好多页面组成的，这个时候就需要通过次标签来实现每个页面之间的链接，来实现网站的页面之间的相互关联的关系。

  • <a>：标签元素最常用的属性：

    标签名 说明 取值 用法
    href 它指示链接的地址，也就是说要打开的页面的地址。 自定义 <a href="http://www.baidu.com">链接的文字内容</a>  这个链接会跳转到百度页面
    target 它指定打开链接地址的目标窗口，也就是说在哪个窗口打开载入连接的页面。

    • _self：当前窗口或者当前框架中打开链接，这个值是默认值，可以不用设置。

    • _blank：浏览器总在一个新打开、未命名的窗口中载入目标文档。

    • _parent：在父窗口中打开载入链接的页面，如果没有父窗口那么在本窗口打开。

    • _top：在顶级窗口中打开载入链接的页面，这个主要是用在多个iframe嵌套的情况下使用。

    		<a href="http://www.baidu.com">链接的文字内容</a>  这个链接会跳转到百度页面
  • 在所有浏览器中，链接的默认外观是：
    • 未被访问的链接带有下划线而且是蓝色的
    • 已被访问的链接带有下划线而且是紫色的
    • 活动链接带有下划线而且是红色的
