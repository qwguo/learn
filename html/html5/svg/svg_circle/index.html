svg绘制圆形的方法：

<circle> 标签可用来创建圆形：

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" style="width:100%; height:150px;">
	<circle cx="50" cy="50" r="40" stroke-width="2" stroke="rgb(255, 0, 0)" fill="rgb(0, 0, 255)" />
	<circle cx="110" cy="60" r="40" stroke-width="2" stroke="rgb(255, 0, 0)" fill="rgb(0, 0, 255)" fill-opacity="0.6" />
	<circle cx="160" cy="70" r="40" style="stroke-width:2px; stroke:rgba(255, 0, 0, .4); fill:rgba(0, 0, 255, .5);" />
	<circle cx="220" cy="70" r="50" transform="scale(.8, .8)" style="stroke-width:2px; stroke:rgba(255, 0, 0, .4); fill:rgba(0, 0, 255, .5);" />
	<circle cx="250" cy="70" r="50" transform="translate(10, 5)" style="stroke-width:2px; stroke:rgba(255, 0, 0, .4); fill:rgba(0, 0, 255, .5);" />
</svg>

圆形的基本定义属性：

<circle>:用来创建圆形。

cx：表示圆形在svg中x轴的位置正数表示向右偏移，负数表示向左偏移。cx是从圆心开始计算的。

cy：表示圆形在svg中y轴的位置正数表示向下偏移，负数表示向上偏移。cy也是从圆心开始计算的。

r：表示圆形的半径。根据此半径来画圆。

使用属性来定义圆形的样式：

stroke-width="2"边线的宽度。

stroke=rgb(255, 0, 0);边线的颜色。

file=rgb(255, 0, 0);矩形填充色。

transform：圆形的变形效果，类似 css 中的 transform

使用css来定义图形的样式

fill：图形内部的填满顏色，类似 css 中的 background-color。

fill-opacity：图形内部的不透明度值，类似 css 中的 opacity，但不包含边框喔！

stroke：图形的边框顏色，类似 css 中的 border-color！

stroke-width：图形的边框宽度，类似 css 中的 border-width！

opacity：图形的不透明度值，类似 css 中的 opacity！

style="fill:rgba(0, 0, 255, .8); stroke-width:2; stroke:rgba(255, 0, 0, .5); height:100; width:100px;"这里包括定义元素的填充颜色，边框，宽高等
