svg绘制折线的方法：

<polyline> 标签可用来创建折线：

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" style="width:100%; height:300px;">
	<polyline points="10,5 30,5 30,25 50,25 50,45 70,45" style="fill:rgba(0,0,0,0); stroke:red; stroke-width:2;" />
	/*折线绘制红十字*/
	<polyline points="100,5 150,5 150,55 200,55 200,105 150,105 150,155 100,155 100,105 50,105 50,55 100,55 100,5" style="fill:red;stroke:red;stroke-width:2;" transform="translate(40)"/>
	/*折线绘制五角星*/
	<polyline points="321 60,271 206,393 115,248 115,370 206 321,60" fill="rgb(255, 120, 0)" fill-rule="evenodd" stroke="rgb(204, 0, 0)" stroke-width="2" />
	/*折线绘制三角形*/
	<polyline points="500,50 400,210 600,210 500,50"  fill="rgb(255, 120, 0)" stroke="rgb(255, 0, 0)" stroke-width="2" />
	/*折线绘制菱形*/
	<polyline points="700,5 620,145 700,275 780,145 700,5" style="fill:rgba(120, 150, 5, 0.5); stroke:blue; stroke-width:2;" />
</svg>

折线的基本定义属性：

<polyline>:用来创建折线。

points：定义由逗号分隔的 x 和 y 坐标对。在本例中，第一个点定义为 10,5，其中 10 是 x 值，5 是 y 值。但是，单独一组点无法在屏幕上显示任何东西，因为这仅告诉 SVG 渲染器从何处开始。在最低限度上，您需要两组点：一个开始点和一个结束点（例如 points="10,5 30,5 30,25 50,25 50,45 70,45”）。

使用属性来定义图形的样式：

stroke-width="2"边线的宽度。

stroke=rgb(255, 0, 0);边线的颜色。

file=rgb(255, 0, 0);矩形填充色。

width：定义矩形的宽度。

height：定义矩形的高度。

transform：矩形的变形效果，类似 css 中的 transform

使用css来定义矩形的样式

fill：图形内部的填满顏色，类似 css 中的 background-color。

fill-opacity：图形内部的不透明度值，类似 css 中的 opacity，但不包含边框喔！

stroke：图形的边框顏色，类似 css 中的 border-color！

stroke-width：图形的边框宽度，类似 css 中的 border-width！

opacity：图形的不透明度值，类似 css 中的 opacity！

style="fill:rgba(0, 0, 255, .8); stroke-width:2; stroke:rgba(255, 0, 0, .5); height:100; width:100px;"这里包括定义元素的填充颜色，边框，宽高等
