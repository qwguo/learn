svg绘制矩形的方法：

<rect> 标签可用来创建矩形，以及矩形的变种：

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" style="width:100%; height:150px;">
<rect x="5" y="15" stroke-width='2' stroke="rgb(255, 0, 0)" fill="rgb(0, 0, 255)"width="100" height="80" />
<rect x="120" y="5" style="fill:rgb(0,0,255); stroke-width:2; stroke:rgb(255,0,0); width:100px; height:100px; fill-opacity:0.6; stroke-opacity:0.5" />
<rect x="240" y="5" rx="10" ry="8" style="fill:rgba(0, 0, 255, .8); stroke-width:2px; stroke:rgba(255, 0, 0, .5); height:100px; width:100px;" />
<rect x="350" y="15" rx="20" ry="10" style="fill:rgba(0, 0, 255, .8); stroke-width:2; stroke:rgba(255, 0, 0, .5); height: 100px; width:200px;" />
</svg>

矩形的基本定义属性：

<rect>:用来创建矩形。

x：表示矩形在svg中x轴的位置正数表示向右偏移，负数表示向左偏移。

y：表示矩形在svg中y轴的位置正数表示向下偏移，负数表示向上偏移。

rx： 圆角的 x 轴半径，类似 css 中的 border-radius，以像素值呈现但是不用带单位。

ry：圆角的 y 轴半径，类似 css 中的 border-radius，以像素值呈现但是不用带单位。

使用属性来定义矩形的样式：

stroke-width="2"边线的宽度。

stroke=rgb(255, 0, 0);边线的颜色。

file=rgb(255, 0, 0);矩形填充色。

width：定义矩形的宽度。

height：定义矩形的高度。

transform：矩形的变形效果，类似 css 中的 transform

使用css来定义矩形的样式

fill：图形内部的填满顏色，类似 css 中的 background-color。

fill-opacity：图形内部的不透明度值，类似 css 中的 opacity，但不包含边框喔！

stroke：图形的边框顏色，类似 css 中的 border-color！

stroke-width：图形的边框宽度，类似 css 中的 border-width！

opacity：图形的不透明度值，类似 css 中的 opacity！

style="fill:rgba(0, 0, 255, .8); stroke-width:2; stroke:rgba(255, 0, 0, .5); height:100; width:100px;"这里包括定义元素的填充颜色，边框，宽高等
