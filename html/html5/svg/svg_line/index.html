svg绘制线条的方法：

<line> 标签用来创建线条：

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" style="width:100%; height:150px;">
	<line x1="0" y1="0" x2="300" y2="120" style="stroke:rgb(99,99,99);stroke-width:2"/>
	<line x1="300" y1="90" x2="400" y2="10" stroke="rgb(99, 99, 99)" stroke-width="2" />
	<!-- 使用线条组合绘制把边线 -->
	<line x1="450" y1="10" x2="503" y2="10" stroke="rgb(255, 0, 0)" stroke-width="2" />
	<line x1="450" y1="10" x2="412" y2="48" stroke="rgb(255, 0, 0)" stroke-width="2" />
	<line x1="503" y1="10" x2="541" y2="48" stroke="rgb(255, 0, 0)" stroke-width="2" />
	<line x1="412" y1="48" x2="412" y2="101" stroke="rgb(255, 0, 0)" stroke-width="2" />
	<line x1="541" y1="48" x2="541" y2="101" stroke="rgb(255, 0, 0)" stroke-width="2" />
	<line x1="412" y1="101" x2="450" y2="139" stroke="rgb(255, 0, 0)" stroke-width="2" />
	<line x1="541" y1="101" x2="503" y2="139" stroke="rgb(255, 0, 0)" stroke-width="2" />
	<line x1="503" y1="139" x2="450" y2="139" stroke="rgb(255, 0, 0)" stroke-width="2" />
</svg>

线条的基本定义属性：

<line>:用来创建线条。

x1：属性在 x 轴定义线条的开始。

y1：属性在 y 轴定义线条的开始

x2：属性在 x 轴定义线条的结束。

y2：属性在 y 轴定义线条的结束。

使用属性来定义圆形的样式：

stroke-width="2"边线的宽度。

stroke=rgb(255, 0, 0);边线的颜色。

file=rgb(255, 0, 0);矩形填充色。

transform：圆形的变形效果，类似 css 中的 transform

使用css来定义图形的样式

fill：图形内部的填满顏色，类似 css 中的 background-color。

fill-opacity：图形内部的不透明度值，类似 css 中的 opacity，但不包含边框喔！

stroke：图形的边框顏色，类似 css 中的 border-color！

stroke-width：图形的边框宽度，类似 css 中的 border-width！

opacity：图形的不透明度值，类似 css 中的 opacity！

style="fill:rgba(0, 0, 255, .8); stroke-width:2; stroke:rgba(255, 0, 0, .5); height:100; width:100px;"这里包括定义元素的填充颜色，边框，宽高等
