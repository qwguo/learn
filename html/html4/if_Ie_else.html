项目 范例 说明
! [if !IE] "逻辑非"运算符
lt [if lt IE 6] 小于运算符
lte [if lte IE 6] 小于等于运算符
gt [if gt IE 6] 大于运算符
gte [if gte IE 6] 大于等于运算符
() [if (lt IE 6)] 用于子表达式，以配合布尔运算符
& [if (lt IE 9)&(gt IE 6) AND运算符
| [if (gt IE 6)|[!IE]] OR运算符
