伪类选择符 Pseudo-Classes Selectors：
  1. a:link{选择符}
  2. a:visited{选择符}
  3. a:hover{选择符}
  4. a:active{选择符}
  5. a:focus{选择符} 单选 多选
  6. :lang(zh-cn){选择符}：lang="zh-cn"中文语言english language
  7. :not(a){选择符}：<span>这样的element结构</span><a>这样的element结构</a>
  8. :root{选择符}：:root这的是根元素，在html文件中也就是HTML本身
  9. :first-child{选择符}：第一个span（被选中）第二个span第三个span第四个span第五个span
  10. :last-child{选择符}：第一个span第二个span第三个span第四个span第五个span（被选中）
  11. span em:only-child{选择符}：第一个span第二个span第三个span第四个span第五个span（被选中）
  12. :nth-child(2){选择符}：第一个span第二个span（被选中）第三个span第四个span第五个span
  13. :nth-last-child(2){选择符}：第一个span第二个span第三个span第四个span（被选中）第五个span
  14. :first-of-type{选择符}：第一个a（被选中）第二个a第一个span（被选中）第二个span第三个span第四个span第五个span
  15. :last-of-type{选择符}：第一个span第二个span第三个span第四个span第五个span（被选中）第一个a第二个a（被选中）
  16. :only-of-type{选择符}：第一个span第二个span第三个span第四个span第五个span（被选中）b第一个a第二个a（被选中）
  17. :nth-of-type{选择符}：第一个span第二个span（被选中）第三个span第四个span第五个span第一个a第二个a（被选中）
  18. :nth-last-of-type{选择符}：第一个span第二个span第三个span第四个span（被选中）第五个span第一个a（被选中）第二个a
  19. :empty{选择符}：第一个span第二个span第三个span第四个span第五个span（他的上一个是空元素）第一个a第二个a
  20. :checked+*{选择符}：第一个span第二个span（选中）第三个span第四个span第五个span（选中）第一个a第二个a
  21. :enabled{选择符}：第一个span第二个span第三个span第四个span（选中）第五个span（选中）第一个a第二个a
  22. :disabled{选择符}：第一个span第二个span（选中）第三个span第四个span第五个span第一个a第二个a（选中）
  23. :target{选择符}：第一个span第二个span第一个a第二个a（单击我）
  24. 这是链接的target目标
