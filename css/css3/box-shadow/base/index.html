box-shadow投影的基本设置：
box-shadow: 1px 1px 5px rgba(0, 0, 0, 0.5);
说明：

box-shadow：用来设置元素的阴影；

  • box-shadow: x y l color;

语法介绍：

  1. x：表示元素的投影在x轴方向的偏移度，取正数表示投影向左偏移，负值表示投影向右偏移。
  2. y：表示元素的投影在y轴方向的偏移度，区正数表示投影向下偏移，负值表示投影向上偏移。
  3. l：表示元素投影的四周扩散程度，取值越高表示扩散程度越大，并且扩散的边缘越模糊。取值为正数。
  4. color：表示元素投影的颜色值。

通过元素投影可以实现好多有趣绚丽的效果。
