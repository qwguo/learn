css3中的filter(滤镜)效果---挤压效果
通过filter:的blur模糊和saturate饱和度
.box-list{width: 1120px; list-style: none; margin: 0 auto; padding: 0; background-color: #000;
  -webkit-filter: blur(15px) saturate(700%);
  cursor: none;
  通过设置transform-style: preserve-3d;来规定子元素将保留其 3D 位置。
  transform-style: preserve-3d;
}
.box-list:after{clear: both; content: ''; display:table;}
.box-list i,.box-list li{width: 100px; height: 100px; border-radius: 100%; }
.box-list i{position: absolute; background-color: #66D9EF; display:none;}
通过设置子元素的z轴偏移来实现3d效果，消除filter的噪点
.box-list li{background-color: #C00; float: left; margin: 20px; transform: translateZ(10px);}
