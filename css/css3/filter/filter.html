css3中的filter(滤镜)效果
  • 正常
  • 灰度：grayscale(0.5);
  • 褐色：sepia(100%);
  • 饱和度：saturate(10);
  • 色相旋转：hue-rotate(180deg);
  • 反色：invert(10%);
  • 透明度：opacity(0.5);
  • 亮度：brightness(0.9);
  • 对比度：contrast(10%);
  • 模糊：blur(1px);
  • 阴影：drop-shadow(10px 10px 10px rgba(1,0,0,.5));
  • 混合

filter：用来这是元素得滤镜效果，是css3的新增属性，它和css only IE有所不同，目前css3中的filter支持的效果有以下10个。

  1. grayscale：灰度（取值为0-1之间的小数）
  2. sepia：褐色（取值为0-1之间的小数）
  3. saturate：饱和度（取值为数字num）
  4. hue-rotate：色相旋转（取值为角度值angle）
  5. invert：反色（取值为0-1之间的小数）
  6. opacity：透明度（取值为0-1之间的小数）
  7. Brightness：亮度（取值为0-1之间的小数）
  8. contrast：对比度（取值为数字num）
  9. blur：模糊（取值为像素值）例如：blur(2px)
  10. drop-shadow：阴影（取值为像素值加颜色）例如：drop-shadow(2px 2px 2px rgba(230, 120, 1, 0.5))

filter:grayscale;设置元素的灰度模式，参数取值为0-1之间或者0%-100%之间，0、0%表示不进行灰度滤镜，1、100%表示完全灰色。
.filter-list li:nth-child(2) img {
-webkit-filter:grayscale(0.5);
-moz-filter:grayscale(0.5);
-o-filter:grayscale(0.5);
filter:grayscale(0.5);
}
filter:sepia;设置元素的褐色度，参数取值为0-1之间或者0%-100%之间，0、0%表示不进行褐色滤镜，1、100%表示最高值褐色滤镜。
.filter-list li:nth-child(3) img {
-webkit-filter:sepia(0.5);
-moz-filter:sepia(0.5);
}
filter:saturate;设置元素的饱和度参数为正数的数字
.filter-list li:nth-child(4) img {
-webkit-filter:saturate(10);
}

.filter-list li:nth-child(5) img {
-webkit-filter:hue-rotate(360deg);
}

.filter-list li:nth-child(6) img {
-webkit-filter:invert(10%);
}

.filter-list li:nth-child(7) img {
-webkit-filter:opacity(0.5);
}

.filter-list li:nth-child(8) img {
-webkit-filter:brightness(0.9);
}

.filter-list li:nth-child(9) img {
-webkit-filter:contrast(10%);
}

.filter-list li:nth-child(10) img {
-webkit-filter:blur(1px);
}

.filter-list li:nth-child(11) img {
-webkit-filter:drop-shadow(10px10px10pxrgba(1,0,0,.5));
}

.filter-list li:nth-child(12) img {
-webkit-filter:grayscale(1) blur(1px) sepia(0.5) saturate(10) drop-shadow(1px1px1pxrgba(1,0,0,.5));
}
