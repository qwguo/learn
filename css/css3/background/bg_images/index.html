元素多背景图片
<div class="bg-div"></div>

.bg-div{
	height: 500px;
	border:3px solid #C00;
	background-image:
		url(/public_image/s1.jpg),
		url(/public_image/s2.jpg),
		url(/public_image/s3.jpg),
		url(/public_image/s4.jpg);
	background-position:
		left top,
		right top,
		left bottom,
		right bottom;
	background-repeat:
		repeat-y,
		repeat-x,
		repeat-x,
		repeat-y;
}

元素设置多背景图片是css3新增的一个属性
