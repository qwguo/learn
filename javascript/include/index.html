页面加载javascript方法

如需在 HTML 页面中插入 JavaScript，请使用 <script> 标签，它可被放置在 HTML 页面的 <body> 和 <head> 部分中。脚本代码必须位于 <script> 与 </script> 标签之间。页面中加载脚本的方法有三种。

  1. <script type="text/javascript" src="a.js"></script>
  2. <script type="text/javascript" >alert("这是javascript弹窗！");</script>
  3. <a href="javascript:alert('这是javascript弹窗！');"></a>
  4. <span onclick="javascript:alert('这是javascript弹窗！');"></span>

其中第3和第4是同样的意思只是在a链接中可以用这种方式来写。

script标签中的属于：

属性名 属性值 描述
async async 规定异步执行脚本（仅适用于外部脚本）。
defer defer 规定当页面已完成解析后，执行脚本（仅适用于外部脚本）。
src URL 规定外部脚本的 URL。
type MIME-type 规定脚本的 MIME 类型。
language script 不赞成使用。规定脚本语言。请使用 type 属性代替它。
charset charset 规定在脚本中使用的字符编码（仅适用于外部脚本）。
xml:space preserve HTML5 不支持。规定是否保留代码中的空白。

  1. async：属性只支持通过src外部引入的脚本，也就是第一种方法。

    <script type="text/javascript" src="a.js" async></script>或<script type="text/javascript" src="a.js" async="async"></script>
    表示脚本异步加载，脚本以下的页面结构不用等待此脚本文件加载完的情况下可以顺利执行，不会出现浏览器停止网页渲染的现象。它和页面的加载是同步执行的。Demo1

    async：是HTML5新增的属性IE浏览器并不支持他，在IE下可以使用defer;

  2. defer：属性是一个布尔属性，属性只支持通过src外部引入的脚本，也就是第一种方法。

    <script type="text/javascript" src="a.js" defer></script>或<script type="text/javascript" src="a.js" defer="defer"></script>
    表示脚本异步加载，脚本以下的页面结构不用等待此脚本文件加载完的情况下可以顺利执行，不会出现浏览器停止网页渲染的现象。它和页面的加载是同步执行的。Demo1

    defer：属性规定是否对脚本执行进行延迟，直到页面加载为止。只有 Internet Explorer 支持 defer 属性。

  3. src：表示引入外部脚本的地址。

  4. type：必选属性，规定脚本的 MIME 类型

    type 属性标识 <script> 和 </script> 标签之间的内容。MIME 类型由两部分组成：媒介类型和子类型。对于 JavaScript，其 MIME 类型是 "text/javascript"。

  5. charset：属性规定在外部脚本文件中使用的字符编码。

    <script type="text/javascript" src="a.js" charset="UTF-8"></script>

    charset 属性与 src 属性一起使用，告诉浏览器用来编码这个 javascript 程序的字符集。它的值是任何一个 ISO 标准字符集编码的名称。
